1. `01_比赛项目收口与完赛标准.md`：总纲与完赛标准
2. `02_演示脚本.md`：演示脚本
3. `03_答辩话术与评审Q&A.md`：答辩与问答
4. `04_素材清单与最终审查表.md`：截图、录屏、简介素材
5. `05_作品简介与亮点包装定稿.md`：对外简介与亮点
6. `06_正式答辩口播版.md`：可直接照读的口播稿
7. `07_最终检查清单.md`：交卷前总检查表
8. `08_提交前收口说明.md`：一句话状态说明
9. `09_与初赛文档的对应关系.md`：与初赛文档的衔接关系

如需确认项目当前状态，优先看 `01`、`07`、`08`。
