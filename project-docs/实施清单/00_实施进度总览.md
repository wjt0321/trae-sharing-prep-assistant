### 关键技术约定（踩坑记录）

1. **后端启动**：用 `npx ts-node src/main.ts` 而非 `pnpm run start:dev`（nest build 不生成 dist 的问题）
2. **PowerShell 限制**：不支持 heredoc（`<<'EOF'`），commit message 用多个 `-m` 参数
3. **git 代理**：push 前设置 socks5h://127.0.0.1:10808，push 后必须 unset（否则影响其他网络操作）
4. **共享包构建**：后端依赖 `packages/shared/dist/`，shared 改动后必须 `pnpm run build`
5. **Prisma 迁移**：新增模型/字段后 `pnpm run db:migrate`，迁移名格式 `YYYYMMDDHHMMSS_描述`
6. **跨模块依赖**：CollaborationModule 被 PlanningModule 和 ExecutionModule import（记录活动流），新增通知模块也应 import CollaborationModule
7. **工作区成员 API**：返回扁平结构 `{ userId, displayName, role, ... }`（不是嵌套 `{ user: {...}, role }`）
8. **Plan.changeReason**：替代从 aiPromptLog 解析 replanReason，replanReason 作为兼容字段保留
9. **@Public 装饰器**：用于公开共享页（无需 JWT），在 `apps/server/src/presentation/` 中定义
10. **前端 API 代理**：`/api/server/*` 通过 rewrites 代理到后端 `http://localhost:4000`
11. **设计系统 token**：在 `apps/web/src/app/globals.css` 的 `@theme inline` 中定义（无 tailwind.config.js）
12. **任务状态机**：TASK_STATUS_TRANSITIONS 定义合法转换，blocked 必填 blockerNote
