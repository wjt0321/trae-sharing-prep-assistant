# 初赛 Demo 功能清单｜灵感到执行 AI 管家

## Demo 目标
在最短时间内做出一个“可感知价值、可交互体验、可直接演示”的完整闭环产品。

## 一、必须完成（MVP）
### 1. 首页输入区
- 输入一个目标 / 想法 / 任务
- 支持补充时间、难度、偏好等条件

### 2. 任务拆解区
- 自动生成任务列表
- 每个任务包含：标题、说明、优先级、建议顺序

### 3. 执行计划区
- 自动生成短周期行动计划（建议 3 天 / 7 天）
- 展示每日重点任务与阶段目标

### 4. 结果输出区
- 输出结构化结果：
  - 待办清单
  - 时间安排
  - 阶段总结
  - 下一步建议

### 5. 二次优化能力
- 用户补充一句新要求后，可重新生成优化版本
- 如：时间更紧、要更偏产品化、需要能演示给评委看

## 二、建议完成（增强版）
### 6. 模板示例
- 比赛项目
- 内容创作
- 学习计划
- 小工具开发

### 7. 可视化呈现
- 阶段流程图
- 时间线
- 卡片式任务展示

### 8. 导出能力
- 导出 HTML 结果页
- 导出简版报告文本

## 三、演示重点
- 不强调“大而全”
- 强调“几分钟内把模糊想法变成可执行方案”
- 强调“输出是结构化结果，不只是聊天文本”

## 四、推荐演示流程
1. 输入：我要参加 AI 创造力大赛，1 周内完成报名和 Demo
2. 展示系统自动拆解的任务树
3. 展示 7 天计划
4. 展示结果页
5. 临时追加要求：更偏产品感、更适合比赛评审
6. 展示系统自动优化后的第二版方案

## 五、验收标准
- 3 分钟内能完成一次完整演示
- 页面结构清楚，评委能一眼看懂
- 至少有 1 个真实场景能让人觉得“我也会用”
- 整个流程体现 AI 真正参与了规划与整理
