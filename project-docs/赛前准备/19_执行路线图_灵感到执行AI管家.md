# 执行路线图｜灵感到执行 AI 管家

## 阶段 1：报名与提案（尽快完成）
- 确认项目名称与一句话定位
- 完成报名创意提案帖
- 选择赛道标签
- 准备项目封面图 / 简介图

## 阶段 2：初赛 Demo 制作
- 明确 MVP 范围
- 设计页面结构
- 完成核心交互流程
- 准备一个最能打动评审的演示场景
- 产出可运行体验链接或 HTML 文件

## 阶段 3：证明材料整理
- 保存关键 AI 对话截图
- 保存任务历史与关键 Session ID
- 保存阶段成果截图
- 整理创作过程说明

## 阶段 4：初赛提交
- 在初赛专区发 Demo 帖
- 补齐演示说明、使用方式、亮点与截图
- 检查链接、页面、文案与合规性

## 阶段 5：复赛预埋
- 记录 Demo 的可扩展点
- 提前规划复赛时可升级的能力
- 预留更完整的用户路径与产品故事

## 当前建议优先级
1. 先把项目定位与提案打磨到能发
2. 再做最小但漂亮的 Demo
3. 最后再补证明材料与展示包装
