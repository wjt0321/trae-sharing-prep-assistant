# 7 天推进节奏表｜灵感到执行 AI 管家

## Day 1：定方向
- 确认项目名、定位、口号
- 确认正式报名帖主版本
- 确认 Demo 主场景与页面结构

## Day 2：定内容
- 打磨创意提案与初赛发帖内容
- 准备 Demo 默认示例数据
- 准备页面核心文案

## Day 3：起页面
- 搭建 Demo 基础页面
- 完成输入区、任务拆解区、执行计划区
- 初步跑通主流程

## Day 4：补结果
- 完成结果输出区
- 增加二次优化能力
- 优化页面层次与模块边界

## Day 5：做质感
- 优化视觉风格
- 统一标题、卡片、按钮和状态样式
- 补示例按钮和空状态文案

## Day 6：做演示
- 按脚本完整演练一次
- 准备截图、录屏、证明材料
- 找出最弱的一处表达并补强

## Day 7：做提交
- 复查报名帖 / Demo / 链接 / 文案
- 准备初赛发帖内容
- 整理所有证明材料并完成提交准备

## 每天都要做的事
- 保存关键过程截图
- 记录关键 Session ID
- 记录页面明显版本变化
