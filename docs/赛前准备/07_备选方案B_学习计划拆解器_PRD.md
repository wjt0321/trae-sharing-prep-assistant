# 备选方案 B｜学习计划拆解器 Demo PRD

## 1. 这是干什么的
这是一个把“我要开始学习某个主题，但不知道怎么安排”自动拆成短期学习计划的小工具。

它不是泛化聊天助手，而是一个学习起步场景下的行动拆解工具。

## 2. 核心场景
用户输入：
- 我想 7 天入门 Python
- 我想 14 天准备一次面试复习
- 我想开始学 AI 绘画，但不知道怎么安排

系统输出：
- 学习阶段划分
- 每天要做什么
- 建议顺序
- 一份可直接执行的学习计划

## 3. 为什么它适合比赛
- 用户场景非常普遍
- 演示闭环清楚
- 输出结构天然好看
- 很容易做出“输入一个目标 -> 输出一套计划”的价值感

## 4. MVP 只做什么
- 输入学习目标
- 自动拆成 7 天或 14 天计划
- 输出每日行动清单
- 输出总体建议

## 5. 不做什么
- 不做题库
- 不做笔记系统
- 不做课程平台
- 不做长期学习社区
- 不做复杂进度管理

## 6. 推荐页面结构
- 顶部标题：学习计划拆解器
- 输入区：输入学习目标
- 计划区：按 Day1 / Day2 / Day3 展示
- 总结区：学习建议 + 注意事项

## 7. 推荐演示输入
“我想 7 天入门 Python，但我不知道该怎么安排。”

## 8. 推荐演示输出
- Day1：了解基础语法
- Day2：变量、条件、循环
- Day3：函数和结构化思维
- Day4：列表、字典等常用结构
- Day5：做一个小练习
- Day6：补齐薄弱点
- Day7：完成一个小 Demo

并输出：
- 每天任务说明
- 推荐顺序
- 一段总结

## 9. 交付要求
- 单页可体验 Demo
- 结构清晰
- 适合截图、录屏、发帖
- 保留关键过程截图和 Session ID

## 10. 成功标准
- 用户能输入学习目标
- 系统能输出清晰计划
- 看起来不像空泛鸡汤，而是真能执行
- 适合初赛展示
