# TRAE AI 创造力大赛｜规则说明

## 赛事信息
- 赛事时间：2026-06-16 00:00 至 2026-08-22 23:59（北京时间）
- 参赛平台：TRAE 官方中文社区
- 参赛工具：仅限 TRAE Work / TRAE IDE
- 参赛形式：初赛仅个人参赛；报名、初赛、复赛均在社区完成；决赛线下路演
- 作品要求：可从 0 到 1 创作，或在参赛期间有明显版本更新

## 赛程
- 报名 / 初赛：2026-06-16 至 2026-07-15
- 复赛：2026-07-21 至 2026-08-09
- 决赛：2026-08-15 至 2026-08-22

## 赛道
- 共 4 个通用赛道 + 1 个附加赛题
- 参赛者需先选择 4 个通用赛道之一提报
- 也可叠加提报附加赛题；若决赛获奖，奖励可叠加
- 选题前应进一步核对各赛道标签、发帖分区和允许的提报方式，避免发错专区或标签

## 初赛要求
- 先在报名专区发“创意提案”帖，添加对应赛道标签
- 报名通过后，在初赛专区发“Demo 作品”帖
- 报名与初赛是同步进行的，但**初赛作品有效前提是报名审核已通过**；如果只发 Demo、报名没过，这个 Demo 不会进入评审
- 官方已单独发布三篇配套必看贴：
  - 报名指南：帮助完成报名、领取报名成功对应激励，并处理“0 门槛报名成功”的流程问题
  - 初赛参赛指南：指导如何发布更容易入围复赛的 Demo 作品帖
  - 保姆级教程：覆盖从生成创意提案到做出可体验 Demo 的完整流程
- 这意味着官方要求的不是“只做功能”，而是“按社区流程完成报名帖 + Demo 帖 + 过程材料”的完整参赛链路
- 报名链路要先完成“生成创意提案，完成报名”，初赛链路要再完成“做出可体验 Demo，提交作品帖”
- 初赛全流程可理解为：报名通过 → 做出可体验 Demo → 按模板发布初赛 Demo 帖（体验链接 + 关键步骤截图 + Session ID）→ 走专业评审通道（可叠加抖音人气通道）→ 公示结果 → 晋级复赛
- 初赛有两条独立晋级通道：
  - 专业评审通道：同账号综合得分最高的 1 个作品晋级
  - 抖音人气通道：同账号人气分最高的 1 个作品晋级
- 同一作品若同时通过两条通道，只占 1 个名额

## 通用要求
- 作品必须全部使用 TRAE Work / TRAE IDE 创作
- 需提供创作过程真实性材料：AI 对话截图、任务历史、关键 Session ID、阶段成果截图等
- 作品原则上应提供可运行体验链接或 HTML 文件
- 报名帖与 Demo 作品帖都不只是“发上去就行”，官方明确在引导参赛者把帖子写完整、写清楚、写得更容易入围
- 报名帖至少应准备：作品名称、赛道选择、创意背景/要解决的问题、核心方案、目标用户或应用场景、为什么想做、作品效果预期等
- Demo 作品帖至少应准备：作品简介、解决什么问题、核心功能、体验方式/链接、关键步骤截图、关键 Session ID、创作过程说明、效果展示、补充说明
- 官方教程明显鼓励先用 TRAE Work 快速生成创意提案 HTML，再把创意逐步落地成可体验 Demo
- 原创性、合规性、提交时限都要严格满足
- 提交后原则上不能擅自补交、替换、撤回或修改
