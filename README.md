# 分享会筹备助手

一个为 `TRAE AI 创造力大赛` 初赛准备的单页 Web Demo，用来把"我要准备一场分享会，但不知道怎么开始"的模糊目标，快速整理成结构化筹备方案。

## 亮点

- 结构化输入：分享主题、听众对象、时长、日期、目标、准备状态六项关键约束
- 轻量规则引擎：根据听众、时长、目标、准备状态动态生成不同侧重的筹备方案
- 4 个阶段任务卡片，每阶段含任务目标与完成标准，并标记建议起点
- 分享结构建议：按时长推荐内容结构与时间分配
- 最终行动清单：前两项标注"先做"，强化执行优先级
- 准备重点与风险提醒：基于听众画像动态生成，更像经验判断
- 页面采用暖中性色工作台风格（Warm Editorial + Minimal Product），避免聊天套壳感

## 推荐演示输入

`我要准备一场小型分享会，但我不知道该怎么安排。`

点击"试试示例"按钮可一键填充完整结构化示例。
